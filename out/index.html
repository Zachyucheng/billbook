Features

为什么选择 Billbook？

一句话记账

「午饭吃了 35 块」「给猫买粮 200」，通过任意 MCP 客户端一句话完成记录。

多对象追踪

按个人、伴侣、宠物、项目等对象分别记账，收支清晰分账。

长期分摊

猫粮分 60 天、订阅分 30 天……自动平摊到每日，消费更真实。

数据分析

分类汇总、周期对比、趋势图表，随时掌握支出结构。

本地存储

所有数据存于本地 SQLite，无需注册账号，无需联网，隐私无忧。

桌面端 + MCP

Electron 桌面应用自带 MCP 服务，浏览器端也可独立使用。

How it works

三步开始 AI 记账

01

安装配置 MCP

克隆仓库、安装依赖，在 OpenClaw、Hermes 或任意 MCP 客户端中配置 billbook 服务。

02

创建消费对象

设置「我自己」「宠物」「项目」等消费对象，为每个对象配置分类和预算。

03

一句话记账

打开你的 MCP 客户端，说「午饭吃了 35 块」，账单自动生成！
